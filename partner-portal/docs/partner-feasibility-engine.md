# Partner Feasibility Engine

The feasibility engine combines:

- Territory profile
- Land size and usable area
- Rainwater capture estimate
- Solar/energy suitability
- Greenhouse footprint
- Crop model
- Buyer readiness
- Partner readiness score

The first version uses manual inputs only. Future API integrations can fetch climate and solar values by latitude/longitude.

All outputs are illustrative and non-binding.
