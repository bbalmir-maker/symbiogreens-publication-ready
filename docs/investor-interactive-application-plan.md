# Investor Interactive Application Plan

The Investor page now includes an interactive analysis module while preserving the existing public investor and partner forms.

## Sections

- Investor Analysis overview: "From First Hub to Caribbean Platform"
- Investment Participation Calculator
- First Hub Economics
- Platform Expansion Upside
- Multi-Hub Revenue Simulator
- Valuation Sensitivity inputs
- Use of Funds
- Risk & Mitigation
- Investor Participation Examples
- Non-binding interest / request review actions

## Compliance Positioning

The Las Terrenas / Northeast Dominican Republic model remains the grounded first-hub case. Platform expansion is presented as illustrative upside only if the first hub is successfully executed. The module avoids guaranteed return language and uses non-binding, discussion-only disclaimers.

## Activation

The calculator works statically with no backend. Supabase persistence and analytics remain optional and feature-flagged through `backend-service.js`.
