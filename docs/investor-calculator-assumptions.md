# Investor Calculator Assumptions

Source data: `assets/js/investor-model-data.js`.

## Current Investment Model

- Target raise: USD 2,200,000
- Investor equity pool: 30%
- Founder Development / Platform Formation Allocation: USD 220,000
- Founder/platform allocation estimate: contribution x 10%
- Estimated investor equity: contribution / 2,200,000 x 30%
- Estimated equity is capped at 30%
- Outputs are illustrative, non-binding, and discussion-only

## Example Checks

- USD 50,000 contribution = 2.27% of raise x 30% = approximately 0.68% estimated equity
- USD 100,000 contribution = 4.55% of raise x 30% = approximately 1.36% estimated equity
- USD 2,200,000 contribution caps at 30% estimated equity

## Scenario Tools

First hub scenarios include conservative, base, and full-potential illustrative cases. Platform tools allow hub count, revenue per hub, EBITDA margin, timeline, and valuation multiple assumptions. None of these are guaranteed return projections.
