# Symbiogreen Publication Ready Documentation

This master package organizes the public website and three connected portals into one maintainable publication-ready system.

## Structure

- `main-website/`: public SymbioGreens / Balponics website.
- `customer-portal/`: customer and buyer product-interest portal.
- `partner-portal/`: strategic partner and regional project portal.
- `investor-portal/`: investor-facing review portal.
- `shared/`: shared configuration examples, approval status constants, and future shared helpers.
- `admin/`: approval workflow notes and email templates.
- `docs/`: project documentation for deployment, QA, and product submission.

## Update Rule

Future updates should be placed in the correct folder:

- Public website update: `main-website/`
- Customer portal update: `customer-portal/`
- Partner portal update: `partner-portal/`
- Investor portal update: `investor-portal/`
- Shared email/form logic: `shared/`
- Admin approval logic/templates: `admin/`
- Documentation: `docs/`
